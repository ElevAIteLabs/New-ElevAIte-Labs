EmailJS Setup Guide — ElevAIte Labs Contact Form

Follow the steps below, then paste your credentials into contact.html inside EMAIL_CONFIG.

Step 1 — Create a free EmailJS account
Go to emailjs.com → Sign Up → free plan allows 200 emails/month.
Step 2 — Add an Email Service
Dashboard → Email Services → Add New Service → choose Gmail (or SMTP) → Connect your hello@elevaitelabs.com account → copy the Service ID.
Step 3 — Create Template 1: Client Confirmation
Dashboard → Email Templates → Create New → Name it template_client_confirm → paste the HTML below → Save.

Step 5 — Get your Public Key
Dashboard → Account → API Keys → copy Public Key.
Step 6 — Paste credentials into contact.html
Open contact.html, find const EMAIL_CONFIG = { and fill in:
  • publicKey — your Public Key from Step 5
  • serviceId — your Service ID from Step 2
  • clientTemplateId — template_client_confirm (exact ID from EmailJS)
  • teamTemplateId — template_team_notify (exact ID from EmailJS)
Note: The Kredoo webhook integration is already live — every form submission will POST to your Kredoo CRM regardless of whether EmailJS is configured. EmailJS emails are an additional layer and will silently skip if the public key is not set.
